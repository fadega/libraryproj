IMPORTANT INFORMATION

All features of the bookstore application depend on one thing : running/executing the folloiwng command.

-php install.php

Here is how we can run this file

Step 1. Locate the file

The file is located inside the 'app' folder. The 'app' folder is inside the 'Project' folder - which is the home directory of the bookstore application. Do this step from the cmd/termianl

Step 2. Execute the file

Now simply type the following in the terminal after changing to the directory where the install.php is located.
php install.php
This step assumes that the environment is configured and ready for php and sqlite.

Step 3. Sign up to access the application

This is another important thing to do before attempting to access the application features
After executing the install.php script, access the index.php file and signup.
